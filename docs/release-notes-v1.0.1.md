# Cadillac Wallpaper Desktop v1.0.1

此版本修复公开下载包无法直接打包的问题。

## 变化

- 将当前支持的足球模板内置到桌面应用资产中。
- 未设置 `CADILLAC_INPUT_ZIP` 时，应用默认使用内置足球模板。
- `CADILLAC_INPUT_ZIP` 仍可用于高级覆盖测试兼容模板。
- 更新中文和英文说明，明确当前只适配足球模板这一款，其他官方主题模板后续更新。

## 发行包

- `CadillacPackager-macos-universal.zip`：macOS Apple Silicon / M 系列和 macOS Intel。

Windows x64 版本后续补发。
